# 📊 Análise de NPS com Python

Projeto desenvolvido durante a iFood Dev Week #2 com foco em análise de dados utilizando Python.

O projeto realiza:

* Leitura de dados CSV com Pandas
* Cálculo do NPS (Net Promoter Score)
* Classificação de promotores, neutros e detratores
* Visualização gráfica do resultado com Matplotlib

## 🚀 Tecnologias utilizadas

* Python
* Pandas
* Matplotlib

## 📈 Funcionalidades

✔️ Importação e tratamento de dados
✔️ Cálculo automático de NPS
✔️ Geração de gráfico visual com zonas de classificação
✔️ Estruturação do código em múltiplos arquivos Python

## 📚 Aprendizados

Durante o desenvolvimento deste projeto, foram praticados conceitos importantes de:

* Manipulação de dados
* Funções em Python
* Visualização de dados
* Organização de projetos
* Análise de métricas

Projeto criado para fins de estudo e prática em análise de dados com Python.
